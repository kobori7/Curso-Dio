O que é linux?

- É um dos sistemas operacionais mais populares e é amplamente utilizado no segmento de servidores e desenvolvedores de software. Ainda existe um grande espaço para crescimento no segmento de computadores pessoais.

- Mascote do linux chama Tux

- O modelo cliente-servidor é uma forma de organizar sistemas de rede em que um computador (cliente) solicita serviços ou informações e outro computador (servidor) responde a essas solicitações.

🔹 Como funciona?

O cliente envia um pedido (ex: abrir um site).

O servidor recebe o pedido, processa.

O servidor devolve a resposta (ex: a página carregada).

👉 Exemplo simples:
Quando você acessa um site pelo navegador, seu computador é o cliente e o computador onde o site está hospedado é o servidor.

🔹 Principais características

✅ Centralização: dados e serviços ficam no servidor.
✅ Organização: cada máquina tem sua função.
✅ Escalável: dá para aumentar servidores conforme cresce o número de usuários.
✅ Mais seguro: controle de acesso pode ser feito no servidor.

🔹 Vantagens

Facilita manutenção (tudo fica concentrado).

Backup mais simples.

Permite vários clientes ao mesmo tempo.

🔹 Desvantagens

Se o servidor cair, o sistema pode parar.

Pode ficar lento se muitos clientes acessarem ao mesmo tempo.

Custo maior para manter servidores robustos.

🔹 Exemplos do dia a dia

Navegar na internet

Apps de banco

Jogos online

E-mail

💡 Resumo final:
👉 Cliente pede → Servidor processa → Servidor responde.
É o modelo que sustenta praticamente toda a internet
