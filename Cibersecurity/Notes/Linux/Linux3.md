Linux virtual box

O Oracle VM VirtualBox é um programa de virtualização — ele permite criar “computadores virtuais” dentro do seu notebook.

👉 Na prática, é como rodar outro sistema operacional sem precisar instalar de verdade.

🧠 O que ele acrescenta na sua máquina

✅ 1. Rodar vários sistemas

Você pode usar:

Ubuntu

Linux Mint

Outra versão do Windows

👉 Tudo ao mesmo tempo.

💡 Muito usado por programadores para testar apps.

✅ 2. Ambiente seguro para testes

Se você:

quebrar o sistema

pegar vírus

apagar algo

👉 Basta deletar a máquina virtual — seu notebook real fica intacto.

✅ 3. Ajuda MUITO quem programa

Exemplos reais:

testar servidores

estudar redes

usar containers

simular ambientes empresariais

👉 É quase obrigatório para quem entra em backend ou DevOps.

⚠️ Mas aqui vai o alerta (IMPORTANTE para seu caso)

Virtualização consome MUITOS recursos.

Cada máquina virtual usa:

👉 RAM

👉 Processador

👉 Disco

Exemplo:

Se seu notebook tem 16 GB:

VirtualBox pode pegar 4–8 GB sozinho 😬

Sobram poucos para o resto.

👉 Isso explica por que sua memória está “quase cheia”.

🔥 Na verdade, o VirtualBox é um dos programas que mais pesam em notebook intermediário.

- shutdown 0: desligar máquina ou para fechar o app, podemos clicar no X e ir na opção salvar o estado da máquina para não perdermos aonde estamos e não precisar pedir o usuario e senha novamente, porém, cuidado.
